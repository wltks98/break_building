# break_building
1-2 C언어및 실습 텀프로젝트

c언어 콘솔에 구현한 간단한 게임입니다
추억의 건물부수기 게임이며 시작시 위에서 건물이 내려오고 캐릭터는 점프와 동시에 부수기 버튼을 누루면 건물을 부수게 됩니다
점수가 10점 오를때 마다 레벨이 상승하고 건물이 내려오는 속도가 증가합니다

점프키:space bar
부수기: f
